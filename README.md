# Grade Clicker - Лабораторная работа №12

## Описание
Каталог игр с обложкой, названием, жанром, описанием и навигацией между играми.

## Цель работы:

Изучить жизненный цикл Activity в Android и научиться правильно
обрабатывать переходы между состояниями. В результате лабораторной работы было
создано интерактивное приложение "Grade Clicker" (Кликер оценок), которое
демонстрирует работу lifecycle callbacks и сохранение состояния при изменении
конфигурации устройства.

## Контрольные вопросы

### 1. Какие три метода вызываются при запуске приложения?

onCreate, onStart, onResume 

### 2. Чем отличается onPause() от onStop()?

onPause() - Activity теряет фокус
onStop() - Activity становится невидимой

### 3. Когда вызывается onRestart()?

Когда происходит перезапуск приложения после Stop

### 4. Что происходит при повороте экрана?

Поворот экрана вызывает изменение конфигурации, Android полностью уничтожает Activity, затем создаёт его заново и из-за этого данные теряются

### 5. В чём разница между remember и rememberSaveable?

rememberSaveable сохраняет значение про рекомпозиции И при изменении конфигурации в отличие от remember

### 6. Почему onCreate() должен вызывать super.onCreate()?

super.onCreate() вызывается для завершения создания Activity

### 7. Для чего используется класс Log?

Класс Log используется для логирования - записывание коротких сообщений в консоль и отслеживания вызова callback-методов

### 8. Какой метод вызывается только один раз за время жизни Activity?

onCreate
